<Example renderer="filterInput" />

## Code

```tsx
<Table
  columns={columns}
  data={data}
  renders={{
    filterInput: ({ props }) => (
      <Input
        className="mt-2.5"
        placeholder={props.placeholder}
        value={props.value}
        onChange={props.onChange}
        {...props}
      />
    ),
  }}
/>
```
