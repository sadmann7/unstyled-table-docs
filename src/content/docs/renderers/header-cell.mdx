## Preview

<Examples renderer="headerCell" />

## Code

```tsx
<Table
  columns={columns}
  data={data}
  renders={{
    headerCell: ({ children, props }) => (
      <th
        {...props}
        className="border-collapse border px-4 pb-3.5 pt-10 text-left text-sm font-bold tracking-wide"
      >
        {children}
      </th>
    ),
  }}
/>
```
