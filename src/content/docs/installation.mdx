---
title: Installation
description: Install the unstyled table adapter using your favorite package manager.
---

<Alert>
  <AlertDescription>
    **Prerequisites**: By defult, the table component is fully unstyled. Bring
    your own styles and feel free to customize the table as you see fit.
  </AlertDescription>
</Alert>

### npm

```sh
npm install unstyled-table
```

### yarn

```sh
yarn add unstyled-table
```

### pnpm

```sh
pnpm add unstyled-table
```
