---
title: bodyRow
description: This component renders the body rows of the table.
---

## Props

The body row renderer receives the following props:

- `children` - The children of the footer renderer.
- `footerGroups` - Similar to the array of [header group](/docs/core-apis/header-group) but for the footer.

## Preview

<Example renderer="bodyRow" />

## Code

```tsx
<Table
  columns={columns}
  data={data}
  renders={{
    bodyRow: ({ children, row }) => (
      <tr
        onClick={() =>
          toast.success(
            `You clicked on ${row.getValue<
              (typeof columns)[number]["accessorKey"]
            >("name")}`
          )
        }
        className="cursor-pointer border-b border-neutral-500"
      >
        {children}
      </tr>
    ),
  }}
/>
```
