title: tableTag
description: This component renders the tags of the table.
---

## Props

The table tag renderer receives the following props:

- `children` - The children of the header renderer. This is the header row.
- `headerGroups` - The array of [header group](/docs/core-apis/header-group). This is useful for creating nested headers.

## Preview

<Example renderer="tableTag" />

## Code

```tsx
<Table
  columns={columns}
  data={data}
  renders={{
    body: ({ children }) => <tbody>{children}</tbody>,
  }}
/>
```
